# HEARTBEAT.md - Periodic Checks

When triggered for a heartbeat check, review these items:

## Checks (rotate through these)

- **Memory:** Are there recent interactions worth persisting?
- **Files:** Any workspace files that need updating?
- **Context:** Anything from recent sessions that should be consolidated?

## When to act

- Something important needs the user's attention
- Memory files need consolidation
- A scheduled task is due

## When to stay quiet

- Nothing new since last check
- Late night hours (respect the user's timezone)
- User is clearly busy
